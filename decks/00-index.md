# 卡组

本部分收录玩家卡组分享与统一格式。

## 卡组分享格式

使用以下格式在玩家之间分享卡组。如果卡组中不包含某类卡牌，可省略该类型的标题与内容。

```
【卡组名称（玩家自定义）】
主阵营：主阵营名
联盟阵营：联盟阵营名 / 无

BOSS
(费用) BOSS卡名

单位
带入数x (费用) 单位卡名

事件
带入数x (费用) 事件卡名

反制
带入数x (费用) 事件卡名
```

## 格式说明

- **卡组名称：** 玩家自定义的卡组名称，便于识别与交流。
- **主阵营：** 卡组所属的主阵营，决定了可使用的卡牌范围。
- **联盟阵营：** 卡组所属的联盟阵营，若无联盟阵营可使用则标注“无”。
- **BOSS：** 卡组中的BOSS卡牌
- **单位：** 卡组中的单位卡牌
- **事件：** 卡组中的事件卡牌
- **反制：** 卡组中的反制卡牌
- **费用：** 卡牌的费用，单位为`绿宝石（e）`，例如`(3e)`表示该卡牌的费用为 3 绿宝石。
- **卡名：** 卡牌的名称，需与游戏中的卡牌名称一致。
- **带入数：** 该卡牌在卡组中的数量，通常为1-4张。
- **注意事项：**
  - 卡组构建需遵守游戏规则，如卡牌数量限制、费用限制等。
  - 卡组构建应考虑卡牌之间的协同与整体策略，以提高卡组的竞争力与实用性。
  - 玩家在分享卡组时应提供足够的卡牌信息，以便其他玩家理解卡组的构建思路与使用方法。
  - 玩家在使用他人分享的卡组时应根据自己的游戏风格与偏好进行适当调整，以发挥卡组的最大潜力。
  - 玩家在分享卡组时应尊重原创，注明卡组的来源与贡献者，以促进社区的交流与合作。
  - 玩家在分享卡组时应提供卡组的构建思路与使用方法，以帮助其他玩家更好地理解与使用该卡组。
