# 第一部分：游戏概述

[返回：规则文档首页](../00-index.md)

## 1.1 游戏目标

两名玩家进行一对一卡牌对战。通过部署单位、使用事件牌，将敌方玩家的生命值从 20 点降至 0 点或以下，即可获得胜利。

## 1.2 游戏配件

- 双方各一套 40 张的卡组
- 记录生命值的工具（纸笔或计数器）

## 1.3 游戏场地

每位玩家拥有以下区域：

- **牌堆**：面朝下放置的卡牌，用于抽牌
- **手牌**：拿在手中可以打出的卡牌
- **部署区**：5 个连续的格子，编号 1 至 5，用于放置单位牌
- **墓地**：使用过的牌、被消灭的单位、弃置的牌放置的区域
