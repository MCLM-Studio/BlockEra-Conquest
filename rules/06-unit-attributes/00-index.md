# 第六部分：单位牌属性

[返回：规则文档首页](../00-index.md)

## 6.1 费用

打出该单位需要消耗的绿宝石数量。

## 6.2 体型

单位占据的部署区格子数量。

| 体型数值 | 占用格子数 |
|---------|-----------|
| 1 | 1 格 |
| 2 | 2 格（必须相邻） |
| 3 | 3 格（必须相邻） |
| 5 | 5 格（占满整个部署区） |

## 6.3 攻击力

单位攻击时造成的伤害数值。部分单位没有攻击力，显示为“—”。

## 6.4 生命值

与[攻血差](../04-combat/00-index.md)有关的数值

## 6.5 类别

单位的分类，决定单位的基本特性和交互规则。

| 类别 | 说明                            |
|------|-------------------------------|
| 友好生物 | 无特殊规则                         |
| 敌对生物 | 无特殊规则                         |
| 建筑 | 除特殊说明外，无法被攻击                  |
| 方块 | 除特殊说明外，无法被攻击                  |
| BOSS | 卡组中只能有 1 张，此卡的阵营将决定卡组的**主阵营** |

## 6.6 阵营

单位的背景归属。

| 阵营 | 类型 |
|------|------|
| 主世界 | 常规阵营 |
| 下界 | 常规阵营 |
| 末地 | 常规阵营 |
| (末地) | 括号阵营，仅在构筑时有效 |

## 6.7 技能

单位拥有的特殊能力。分为**被动技能**和**主动技能**。
