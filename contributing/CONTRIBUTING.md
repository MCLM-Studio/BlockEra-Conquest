# 贡献指南

感谢你为《方块纪元：征伐》官方开源 Wiki 贡献内容。本指南用于统一协作流程与文档风格，确保内容可读、可维护、可版本化。

## 贡献范围

- 规则修订：澄清、补充或纠错现有规则。
- 新增内容：新增技能、事件牌、单位牌数据等条目。
- 结构调整：目录重构、索引整理、命名规范优化。

## 基本流程

1. 确定修改范围与目录位置。
2. 编辑对应 Markdown 文件。
3. 同步更新索引：`SUMMARY.md` 与所在目录的 `00-index.md`。
4. 自检链接、标题与编号。

## 变更说明要求

- 规则修订：说明修改动机与影响范围。
- 新增内容：标注来源或依据（如内部设计稿、版本变更说明）。
- 结构调整：说明目录变更原因，并确保索引同步。

## 文档与命名规范

- 文件命名：`NN-名称.md`，`NN` 为两位编号。
- 索引文件：每个目录使用 `00-index.md` 作为入口。
- 标题规范：标题层级不跳级，一级标题仅用于页面标题。
- 术语一致：同一术语保持统一译名与定义。

## 自检清单

- `SUMMARY.md` 是否包含新增页面。
- 目录 `00-index.md` 是否已更新。
- 链接是否存在断链或路径错误。
- 文件编号是否与索引顺序一致。
- 术语是否与词典或已有条目保持一致。
