# 编辑风格指南

本文档定义本 Wiki 的写作与排版规范，帮助保持整体风格一致。

## 写作原则

- 清晰：优先使用简洁句式，避免歧义。
- 一致：术语、名称、格式保持统一。
- 可检索：标题与关键术语使用稳定命名。

## 标题与结构

- 一级标题仅用于页面标题。
- 标题层级不跳级（`##` 之后为 `###`）。
- 建议结构：概述 → 规则/效果 → 条件/限制 → 例子 → 备注。

## 术语与命名

- 术语应与其他条目保持一致。
- 特殊机制或关键词可加粗。

## 数值与单位

- 数值与单位之间加空格（如“增加 3 点伤害”）。
- 范围表述使用“至”（如“ 1 至 3 ”）。
- 概率统一为百分比（如“20%”）。

## 标点与排版

- 中文为主，标点使用【中文全角】。
- 列表项保持同一语法结构。
- 超过 2 行的说明建议使用列表。

## 链接与引用

- 相关条目使用相对路径链接。
- 目录入口页统一为 `00-index.md`。
